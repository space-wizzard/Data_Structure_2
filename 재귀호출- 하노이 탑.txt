순차 자료구조 
장점: 접근이 빠르다.
메모리 저장 방식- 논리적인 순서와 물리적인 순서가 일치
			 메모리 저장 시작 위치부터 빈자리 없이 자료를 순서대로 연속하여 저장
 연산 특징 - 삽입, 삭제 연산을 해도 변경된 논리적 순서와 물리적 순서가 일치
프로그래밍 기법 - 배열을 이용한 구현

연결 자료구조
메모리 저장 방식- 논리적인 순서와 물리적인 순서가 불일치
			 메모리 저장 위치나 물리적 위치 상관없이 링크로 위치를 저장한다.
연산 특징 - 삽입, 삭제 연산을 해도 링크 정보만 변경되고 물리적 순서는 변경된다.
프로그래밍 기법 - 배열을 이용한 구현

2차원 배열의 원소 위치 계산 방법 = 공식 암기
다항식 1,2차 배열로 저장하는 방법

연결리스트 삽입 삭제 함수
원형연결리스트 -> null인 경우가 없으니까(마지막 노드가 첫번째 노드로 이어짐) head인 경우 루프가 종료된다.
이중연결리스트 -> 삽입 삭제
